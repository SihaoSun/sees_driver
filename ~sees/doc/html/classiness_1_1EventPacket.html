Factory function to create an event packet. The events should all have the same overflow timestamp. Use iness::aedat::getEventTs(iness::time::TimeUs) and iness::aedat::getOverflowTs(iness::time::TimeUs) to separte a 64bit stamp into the upper and lower part.

Parameters
_eventsVector of event. Will be copied.
