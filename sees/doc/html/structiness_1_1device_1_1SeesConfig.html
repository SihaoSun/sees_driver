 Event threshold - Controls the sensitivity of the device to changes. Currently this is a directly mapped out sensor setting and we do not yet have a relation between the change in light falling on a pixel that triggeres an event and its value. Values in [25,150] are possible. Changing this value may affect the balance between on and off events.

 Event balance - Controls the ratio between on and off events. Currently this is a directly mapped out sensor setting and we do not yet have a relation as to how exactly it influences the ratio for a given intensity step. Values in [0,31] are possible.
