Detailed Description

template<typename T_MAP_TYPE, typename T_CRTP_CHILD, typename T_ENCAPSULATED_ITERATOR_TYPE>
class iness::util::StdMapIteratorBase< T_MAP_TYPE, T_CRTP_CHILD, T_ENCAPSULATED_ITERATOR_TYPE >

These are definitions for templated iterators that allow direct access to second pair members of std::map<...> iterators, that is they circumvent the need for access through ->first and ->second.
