Detailed Description

Defines requirements for the device that is to be loaded (-1 indicates no requirement for a given entry)

Our Sees device interface class can connect to multiple types and will connect to the first one it finds and supports. In order to restrict it to a specific device or device type, requirements can be specified through this struct. Our Sees device interface class will then connect to the first device fulfilling all specified requirements and to none if none fits all.
